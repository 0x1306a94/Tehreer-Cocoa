  • openTypeTag

    Returns the OpenType tag of the specified script.

    The association between Unicode Script property and OpenType script tags is taken from the specification: https://docs.microsoft.com/en-us/typography/opentype/spec/scripttags.

    If more than one tag is associated with a script, then the latest one is returned. For example, Devanagari script has two tags, deva and dev2. So in this case, dev2 will be returned.

    If no tag is associated with a script, then DFLT is returned.
